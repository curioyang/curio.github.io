йҰ–е…ҲиҜҙжҳҺдёҖзӮ№paged attentionеҸӘжҳҜдёҖз§Қattentionдёӯзҡ„kvCacheз®ЎзҗҶж–№жЎҲпјҢе№¶дёҚжҳҜж–°зҡ„attentionи®Ўз®—ж–№ејҸгҖӮ

PagedйЎҫеҗҚжҖқд№үжҳҜдёҺж“ҚдҪңзі»з»ҹдёӯзҡ„еҲҶйЎөз®ЎзҗҶжңүе…ізі»пјҢpaged attentionе°ұжҳҜе°ҶkvCacheиҝӣиЎҢеҲҶйЎөз®ЎзҗҶпјҢиҮідәҺдёәд»Җд№ҲиҰҒиҝӣиЎҢеҲҶйЎөз®ЎзҗҶе°ұйңҖиҰҒе…ҲеӣһйЎҫдёҖдёӢLLMдёӯзҡ„kvCacheзҡ„зү№зӮ№гҖӮ
еңЁйҰ–ж¬Ўprefillйҳ¶ж®өпјҢkvcacheзҡ„еӨ§е°ҸжҳҜ0пјҢд№ҹе°ұжҳҜиҫ“е…ҘеҲ°жЁЎеһӢзҡ„жҳҜдёҖдёӘз©әзҡ„tensorпјҢеҪ“з„¶д№ҹеҸҜд»ҘдёҚиҫ“е…Ҙз»ҷжЁЎеһӢпјҲpsпјҡиҝҷе°ұйңҖиҰҒжЁЎеһӢж”ҜжҢҒдёҚзЎ®е®ҡж•°йҮҸзҡ„иҫ“е…ҘиҠӮзӮ№пјҢиҝҷеңЁpythonд»Јз ҒдёӯжҳҜеҫҲе®№жҳ“зҡ„пјҢдҪҶжҳҜеңЁз«Ҝдҫ§и®ҫеӨҮдёӯеӨҡж•°жҳҜдёҚж”ҜжҢҒеҸҜеҸҳж•°йҮҸзҡ„иҫ“е…Ҙзҡ„пјүпјҢprefillйҳ¶ж®өз»“жқҹеҗҺдјҡдә§з”ҹprefillиҫ“е…Ҙзҡ„tokensж•°йҮҸ+1дёӘзҡ„kvCacheпјҢиҖҢжҜҸж¬Ўиҫ“е…Ҙзҡ„tokensж•°йҮҸе№¶дёҚеӣәе®ҡпјҢйӮЈд№ҲеҰӮдҪ•з®ЎзҗҶkvCacheзҡ„еҶ…еӯҳе‘ўпјҹ

  1. йқҷжҖҒеҲҶй…Қпјҹ kvCacheзҡ„еӨ§е°ҸжҳҜдјҡдёҚж–ӯеўһй•ҝзҡ„пјҢйқҷжҖҒеҲҶй…ҚжҳҜзҡ„еӨ§е°Ҹи®ҫзҪ®дёәеӨҡе°‘йғҪдёҚи§Ғеҫ—еҗҲйҖӮпјҢеӨӘеӨ§дәҶдјҡеҚ з”ЁиҝҮеӨҡеҶ…еӯҳпјҢдё”еҸҜиғҪеҮәзҺ°еӨ§йғЁеҲҶз”іиҜ·зҡ„еҶ…еӯҳе№¶жІЎжңүеҲ©з”Ёзҡ„жғ…еҶөпјҢе°ҸдәҶе°ұдёҚеӨҹеӨҡиҪ®еҜ№иҜқдә§з”ҹзҡ„kvcachekvcacheеӯҳеӮЁгҖӮ пјҲXпјү
  2. еҠЁжҖҒеҲҶй…ҚпјҹеҠЁжҖҒеҶ…еӯҳзӣёеҜ№жқҘиҜҙиҫғдёәеҗҲйҖӮпјҢдҪҶжҳҜд»ҺзҺ°еӯҳзҡ„ж“ҚдҪңзі»з»ҹзҡ„еҶ…еӯҳз®ЎзҗҶжқҘзңӢпјҢеҶ…еӯҳзўҺзүҮй—®йўҳжҳҜдёҖдёӘеҫҲйҡҫи§ЈеҶізҡ„й—®йўҳпјҢиҝҷеҜ№дәҺLLMжҺЁзҗҶжқҘиҜҙд№ҹжҳҜдёҖдёӘйә»зғҰпјҢkvCacheзҡ„еӨ§е°ҸдјҡеҜјиҮҙеңЁеӨҡж¬ЎеҜ№иҜқеҗҺпјҢжІЎжңүеҗҲйҖӮзҡ„иҝһз»ӯз©әй—ҙжқҘеҲҶй…Қз»ҷkvCacheпјҲзўҺзүҮеҢ–зҡ„еҶ…еӯҳдјҡеҪұе“Қиҝҷз§Қиҝһз»ӯеҲҶй…Қзҡ„ж–№ејҸпјү
